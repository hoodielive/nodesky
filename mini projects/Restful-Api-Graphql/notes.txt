Routing 
  Flexible pattern matching 
  Handles parameters 
  Multi router support 
  Static & Dynamic configuration 
  Support for all HTTP verbs 
  Order based -> order DOES matter!

    app.get('*', (req, res) => res.json({ ok: true }));
